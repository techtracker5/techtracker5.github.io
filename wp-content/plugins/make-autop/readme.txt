Allows you to apply wpautop() function to all posts and pages

== Description ==

After installing Dean's FCKEditor for Wordpress you will need to apply wpautop() to all posts and pages

После установки Dean's FCKEditor for Wordpress необходимо выполнить wpautop() для всех записей блога и страниц

== Installation ==

1. Upload `make-autop` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the «Plugins» menu in WordPress
1. Go to plugin settings page
1. Click the only button `Apply wpautop()`

---

2. Загрузите директорию `make-autop` в директорию `/wp-content/plugins/` на вашем сайте
2. Активируйте плагин в меню «Плагины»
2. Перейдите на страницу настроек плагина
2. Нажмите единственную кнопку `Применить wpautop()`
