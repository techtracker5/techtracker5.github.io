How to auto save & send files from Telegram to Dropbox for free.

Posted on November 18, 2021  •  6 minutes  • 1237 words

Do you want to auto save and send all your telegram files either available on Telegram channel or group to your dropbox account? If yes then there is not even 1 bot available to do that but still it is possible to send all your telegram files to dropbox account via third party automation service available on internet which can integrate telegram and dropbox using webhook.

Telegram provides unlimited cloud storage with 2GB file size limit so millions of users store thier files in it but telegram don’t give you direct link for the files and telegram is not reliable to use it as backup platform as if you don’t login in your telegram account atleast once in year then it will be deleted and all your data stored will be lost which can get you personal and financial losses, 

so, it is very important to save your files in numerous reliable backup platform so you won’t get issues later, 

In Telegram, there are alot of channels and groups where they share different types of  files and it is also well utilised by teachers to create groups and collect assignments from students, but the problem is if they want to backup the assignment or files to reliable backup platform like Dropbox then they have to download each assignment or file and upload it to dropbox which will use alot of network data and tiresome, Isn’t it amazing if we can auto save telegram files to dropbox? It will save valuable time.

In this scenario, we have a workaround we found a easy to use automation platform named pabbly where you can integrate multiple applications and automate tasks, In pabbly you can integrate telegram with dropbox to auto save files from telegram channel or group to dropbox account for free, so do you like it? are you interested in pabbly? If yes then let’s know little more info before we sign up and get started.

- You can do manual email signup or use Google signup for automatic login.

- Once you login, Tap on Access Now

**<img src="https://lh3.googleusercontent.com/-U6eEC7ThXiE/YZaTYUqP6GI/AAAAAAAAHYQ/dWlFinuqhmcyUHa17q8de2XzpGNmoLIsACLcBGAsYHQ/s1600/1637258044648415-5.png" alt=""> **

- Tap on + Create Workflow

**<img src="https://lh3.googleusercontent.com/-0Fmyt73qStk/YZaTPNeUqNI/AAAAAAAAHYE/4KVVTRxY028uYClka_goEi3mvi7VHtZVgCLcBGAsYHQ/s1600/1637258029191668-6.png" alt=""> **

- Enter workflow name and tap on Create

**<img src="https://lh3.googleusercontent.com/-RQCxgZw693g/YZaTLIvkPiI/AAAAAAAAHX8/T3G0EJfpB20mWP6t9I8GoqyXBbbOyUcCwCLcBGAsYHQ/s1600/1637258014646754-7.png" alt=""> **

- In Choose App, Enter Telegram and Tap on Telegram Bot.

**<img src="https://lh3.googleusercontent.com/-g2WLA1iqNvc/YZaTHtcTlLI/AAAAAAAAHX4/Mqx9zlFCGaAIj1CCIhDWeacXqoRniVDZACLcBGAsYHQ/s1600/1637258007701012-8.png" alt=""> **

- In Trigger Event, Select : Set Webhook / Watch Updates and Tap on Connect.

<img src="https://lh3.googleusercontent.com/-_ydlnb5wri4/YZaTF_8y-iI/AAAAAAAAHXs/83-1w9hf2wkpxJ3xjM8PDRiG9xu5v456wCLcBGAsYHQ/s1600/1637257999052257-9.png" alt="">

- Select : Add New Connection and Scroll down. It’s time for Telegram you need to generate a bot token.

<img src="https://lh3.googleusercontent.com/-3msvKpDMkB4/YZaTDryH75I/AAAAAAAAHXo/-FvFIah88S0tP-u0yogga9dJ4deoyBNHACLcBGAsYHQ/s1600/1637257989460705-10.png" alt="">

- Go to Botfather telegram bot, and tap on START

**<img src="https://lh3.googleusercontent.com/-SRiwfWEfrI0/YZaTBNjjuMI/AAAAAAAAHXk/y0CUjIYl1pY_grZT_FrU6vCLXcdb6Av3QCLcBGAsYHQ/s1600/1637257975704244-11.png" alt=""> **

- Enter and send /newbot command, it will ask you choose a name for bot.

<img src="https://lh3.googleusercontent.com/-Jvl5VwV-9gE/YZaS9vR2hEI/AAAAAAAAHXc/gbC4yihYhaAkJhv8nYU1eZkczxu05c4bgCLcBGAsYHQ/s1600/1637257964523291-12.png" alt="">

- Enter and send your desired bot name, now it will ask you send username.

<img src="https://lh3.googleusercontent.com/-ysvLevsrqxs/YZaS65PiGWI/AAAAAAAAHXU/okosfnuMJm0aKTZgUCERUKXbhXibmIYbgCLcBGAsYHQ/s1600/1637257951972631-13.png" alt="">

- Send username to botfather with extension Bot or bot for example : TD_Upload_Bot.

- You will get bot token, just tap on it to copy, once you copied go back to Pabbly.

<img src="https://lh3.googleusercontent.com/-BTtQZKfxWs8/YZaS36i7x7I/AAAAAAAAHXI/zV2fF8iqWVATdjOU5sKcHsM64PZS_ONigCLcBGAsYHQ/s1600/1637257944365577-14.png" alt="">

- Paste your bot token in blank and tap on Save to continue further.

<img src="https://lh3.googleusercontent.com/-ahlOj2zIDBI/YZaS17ShHfI/AAAAAAAAHXE/OS4qTuO38Ywarj3Z9gnUNZTPABFhiz2VQCLcBGAsYHQ/s1600/1637257933763806-15.png" alt="">

- Now, go back to telegram and create a new group or channel with name of your choice and tap on ADD MEMBERS

<img src="https://lh3.googleusercontent.com/-8o24o30zMpU/YZaSzUbhz4I/AAAAAAAAHXA/oNzmL44BImMCnWJajHfetDcyF1oR1bi7wCLcBGAsYHQ/s1600/1637257928487864-16.png" alt="">

- In search, search for the bot that you created earlier in botfather, once found just tap on it.

<img src="https://lh3.googleusercontent.com/-En1DNf9VLAU/YZaSx4YqPAI/AAAAAAAAHW4/oQsg4KaLYco3UtNxZk_B_l2Sm3XvK176wCLcBGAsYHQ/s1600/1637257922108917-17.png" alt="">

- Tap on ✓

**<img src="https://lh3.googleusercontent.com/-wpNkPJTkgVU/YZaSwRVJFOI/AAAAAAAAHWw/eiyXRimsBX8QoTnG6UPUuxf5dhcHMfacgCLcBGAsYHQ/s1600/1637257915580808-18.png" alt=""> **

- Tap on ADD

**<img src="https://lh3.googleusercontent.com/-TCX7wvZIUgk/YZaSu3RP0KI/AAAAAAAAHWs/oKSMyP2OT10-XxwAYv4Sqppp89OAZk5eQCLcBGAsYHQ/s1600/1637257908023912-19.png" alt=""> **

- Once, bot added tap on it.

<img src="https://lh3.googleusercontent.com/-NlRxVferYIw/YZaSs00gUFI/AAAAAAAAHWo/uzusG49rliMvkv0TFXDzMNxZGn-rVsQewCLcBGAsYHQ/s1600/1637257901840231-20.png" alt="">

- Tap on **Promote to admin **

**<img src="https://lh3.googleusercontent.com/-QWr5JPZI78c/YZaSrdBYawI/AAAAAAAAHWk/wdZWHOYiC38g_zHldWU7KS2xl8fYDCWLQCLcBGAsYHQ/s1600/1637257894625803-21.png" alt=""> **

- Enable all except last two, and in custom title enter : Admin then tap on ✓

**<img src="https://lh3.googleusercontent.com/-lcOPKPsfNtE/YZaSpq_YaII/AAAAAAAAHWc/PNu2H8suoA8By21wqkisV_a-QdJflBuuQCLcBGAsYHQ/s1600/1637257883061920-22.png" alt=""> **

- Done, now send any file in group or channel under 20MB size.

- Go back to Pabbly.

<img src="https://lh3.googleusercontent.com/-SdHDAfyyDp8/YZaSmiE_PII/AAAAAAAAHWY/rQfO-kuqGJQch6siSX5lB1v5lMkPq3klwCLcBGAsYHQ/s1600/1637257874274873-23.png" alt="">

- Tap on Save & Send Test Request, you will get all details of group using bot,

- In response, scroll down and note down the following sections in notepad it will be  used in future.

- Message Document File Name

- Message Document Mime Type

- Message Document File Id

- Scroll down to create another trigger.

<img src="https://lh3.googleusercontent.com/-waa-dlJsqJ8/YZaSkc0b8tI/AAAAAAAAHWQ/VTG6VTyKPjsVmjf2QUzxfC5-VCm1J5dngCLcBGAsYHQ/s1600/1637257867783571-24.png" alt="">

- In Choose App : Enter Telegram and Tap on Telegram Bot.

- In Action Event, Select : Get File and Tap on Connect.

<img src="https://lh3.googleusercontent.com/-2NDwE0Z9B84/YZaSg91EJgI/AAAAAAAAHWI/JJ1X6-GUR68xzDuzbdkIrxnH5rr6prfawCLcBGAsYHQ/s1600/1637257852697361-26.png" alt="">

- Tap on Select Existing Connection, then tap on Save

**<img src="https://lh3.googleusercontent.com/-rP0F58HQ4BM/YZaSfBiGq1I/AAAAAAAAHWE/q7jBForxEMwcryBBdpPzwthlMk4fL0MkQCLcBGAsYHQ/s1600/1637257840909089-27.png" alt=""> **

- In File ID, select : 1. Message Document File Id : xxxxxxxxxx——-xxxxxxxxx and Tap on Save & Send Test Request.

**<img src="https://lh3.googleusercontent.com/-FC7UK2BppmU/YZaScI6WQyI/AAAAAAAAHV8/x-HtNyVK-1o22BLopjFuldoORNYH6yNUACLcBGAsYHQ/s1600/1637257832324362-28.png" alt=""> **

- You will get response, 

- 

In response, just note  : Result File Path and scroll down to create another trigger.

<img src="https://lh3.googleusercontent.com/-n5QhnJB1brQ/YZaSZ1lmVgI/AAAAAAAAHV4/Svh6SWfRwugXn35vdMdiq8CTrhfoGFk9QCLcBGAsYHQ/s1600/1637257820203144-29.png" alt="">

- In Choose App : Enter Dropbox and Select Dropbox

**<img src="https://lh3.googleusercontent.com/-OP26wOa-xvU/YZaSW7b_mOI/AAAAAAAAHV0/3xjPWtWzZC48HC_HlBs3KxYt_Z498UB_QCLcBGAsYHQ/s1600/1637257813629778-30.png" alt=""> **

- In Action Event, Select : Upload File then tap on Connect.

- Select : Add New Connection and Tap on Connect With Dropbox.

- Now, you have to authenticate Dropbox with Pabbly.

- Once, Authentication done just simply tap on Save

**<img src="https://lh3.googleusercontent.com/-zbG5FwcOkx8/YZaSTP112sI/AAAAAAAAHVs/3VpRrM_enZADR-xfJN1JBh6M6ebydFiBgCLcBGAsYHQ/s1600/1637257794489806-32.png" alt=""> **

- https://api.telegram.org/file/bot : copy & paste your bot token here with / at end.

- Just after that Select : Result File Path

- In folder path, Enter your dropbox folder name with / at first, example : /files.

- Tap on Save & Send Test Request

Perfecto, You successfully integrated Telegram and Dropbox to automate tasks.
